# Mood Governance

TailwindScore 的 preset 方向是 governed premium commerce presets。

不是：

- multiple theme personalities
- template marketplace
- one preset one theme

## 1. Mood Variation, Not Identity Variation

以下可以变化：

- contrast temperament
- accent energy
- spacing calmness
- motion softness
- shell pacing
- content tone

以下不能变化：

- component taxonomy
- page architecture
- purchase flow mechanics
- archive logic
- checkout logic

## 2. Preset Intent

当前基础 presets：

- `minimal-editorial`
- `premium-dtc`
- `soft-luxury`
- `modern-lifestyle`
- `dark-commerce`

它们都属于同一主题人格，只是在同一系统内调整 commerce mood。

## 3. Long-Term Standard

新增 preset 必须回答：

1. 它是不是仍然属于 TailwindScore 的设计语言
2. 它是否依然消费同一 token system
3. 它是否仍然复用同一 runtime
4. 它是否没有逼出新的组件树

如果不能满足，就不应该进入 preset registry。
