# Post-Purchase Language

- Use restrained, service-oriented language rather than motivational or gamified copy.
- Empty states should reassure and orient, not sell aggressively.
- Prefer customer terms like "Orders", "Addresses", and "Account details" over system language.
- Keep intros short and editorial in tone: clear, quiet, and helpful.
- Unsupported patterns for this stage: loyalty dashboards, points systems, SaaS-style activity feeds, CRM summaries.
