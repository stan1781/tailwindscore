# Language Philosophy Enforcement

## Allowed Language Philosophy

Allowed:

- calm reassurance
- editorial commerce tone
- soft premium support language
- globally adaptable messaging

## Disallowed Language Philosophy

Disallowed:

- conversion urgency
- fake scarcity
- SaaS-style UX language
- AI assistant tone
- hype DTC phrasing
- aggressive marketing copy

## Enforcement Heuristic

If a line sounds like:

- pressure
- novelty theater
- productized app onboarding
- assistant-style handholding

it should not ship as part of TailwindScore commerce language.

If a line sounds like:

- clear support
- calm guidance
- quiet confidence
- globally translatable reassurance

it is within the allowed philosophy.
