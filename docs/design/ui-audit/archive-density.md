# Archive Density

## Goal

Keep archive cards breathable and catalog-led while preserving action clarity.

## Decisions

- Product cards now share the card radius token and use surface shadow only as a quiet base layer.
- Card body/footer spacing is widened slightly to improve title/price/action rhythm.
- Archive swatches use the same swatch radius language as PDP swatches.
- Action reveal is opacity-led instead of movement-led.
- Collection row gap is increased to reduce marketplace density.

## Replaced Patterns

- Tight footer padding replaced with calmer card spacing
- Smaller archive swatches replaced by slightly more legible chips
- Motion-led reveal replaced by opacity-led reveal

## Deprecated Styles

- Dense marketplace card pacing
- Overexpressive action reveal
- Radius drift between archive and PDP swatches

## Unsupported Interactions

- Hover lift as product-card identity
- Card chrome stronger than media hierarchy
- Action rows that visually jump the card layout
