# Documentation Health

## Metrics

- Active docs count: number of files inside the governed active IA used for current work
- Archive ratio: archived markdown count divided by total markdown count
- Duplicated governance ratio: governance concepts maintained in more than one active canonical file
- AI context size: number of default entry files AI must read before implementation
- Lifecycle coverage: whether active, reference, and archive states are explicitly defined
- Orphan docs: files with no clear owner or lifecycle
- Superseded docs: files still active after replacement
- Unresolved duplication: known duplicate clusters not yet merged
- Frozen legacy count: number of explicitly frozen legacy areas
- Peer directory count: total number of top-level `docs/` peer directories
- Canonical overlap count: number of known ownership-overlap clusters still requiring reduction
