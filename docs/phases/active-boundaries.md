# Active Boundaries

## Allowed

- `docs/`
- critical clarification
- governance correction
- localization clarification
- operational updates
- lifecycle status updates
- archive maintenance

## Forbidden

- large-scale docs restructuring
- new governance families
- new governance abstractions
- new narrative governance docs
- canonical workflow rewrites
- new AI entry layers
- parallel authority
- new documentation architecture
- new commerce features
- new runtime modules
- new scanner types

## Freeze Escalation Rule

Break the stability freeze only when one of these is true:

- governance critical leak
- architecture drift
- localization governance failure
- runtime governance inconsistency
- canonical authority regression
