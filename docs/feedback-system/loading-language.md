# Loading Language

## Principle

Loading language should describe the customer's current task in plain commerce terms.

## Preferred Phrases

- `Adding to bag`
- `Updating bag`
- `Updating selection`
- `Checking availability`

## Avoid

- technical verbs like `processing request`
- vague copy like `please wait`
- brand theater or editorial flourish during high-friction steps

## Behavior

- Loading states may dim surrounding controls.
- The active area should receive `aria-busy="true"`.
- A matching live-region update should announce the operation once.
