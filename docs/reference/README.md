# Reference Boundary

`docs/reference/` is the governed home for human-first lookup material.

## What Belongs Here

- tokens
- surface maps
- component maps
- preset maps
- registry maps
- stable lookup indexes

## Transitional Rule

Most existing topical directories under `docs/` are still in transitional legacy locations. During this phase:

- they are frozen as human reference
- they are not part of default AI context
- they should be re-homed here only when a consolidation pass has a clear canonical destination

## Legacy Freeze Rule

Frozen legacy directories must declare:

- frozen status
- migration target
- canonical replacement
- AI non-default warning
