# Mobile Navigation

## Goal

Mobile navigation should support the commerce journey without becoming the whole experience.

## Supported Behavior

- right-side slide drawer
- focus trap while open
- body scroll lock
- escape to close
- backdrop click to close
- nested submenu toggles
- reduced-motion friendly transitions

## Motion Rules

- Use smooth, short transitions.
- Keep translation distance modest.
- Avoid bounce, spring, or theatrical motion.
- Preserve clarity even when reduced motion is enabled.

## Ergonomics

- Primary links stay large and easy to scan.
- Submenu toggles remain distinct from links.
- Utility actions sit in a separate footer region inside the drawer.
- Keep hierarchy readable before it becomes deep.

## Unsupported Behavior

- app-style persistent bottom tabs
- full-screen route transitions
- gesture-heavy drawer mechanics
- oversized promotional panels inside the drawer
