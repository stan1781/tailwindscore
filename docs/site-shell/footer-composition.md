# Footer Composition

## Goal

The footer should extend the editorial commerce shell with calm hierarchy and predictable scanning.

## Core Regions

- brand summary
- commerce links
- support links
- editorial links
- newsletter slot
- social slot

## Composition Rules

1. Footer groups should read as support structure, not as a sitemap dump.
2. Link lists remain concise and typographically quiet.
3. Newsletter and social areas are optional slots, not required placeholders.
4. Mobile layout should stack cleanly without card-like treatment.

## Rhythm

- use shared spacing tokens
- keep headings small and clear
- keep link density lighter than marketplace themes
- preserve larger separation between grid and legal row

## Unsupported Behavior

- giant footer directory structures
- promotional banner footers
- crowded badge stacks
- decorative card sections inside the footer
