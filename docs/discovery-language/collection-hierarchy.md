# Collection Hierarchy

## Goal

Collection browsing should maintain a stable hierarchy across archive and search surfaces.

## Hierarchy order

1. Collection label or search context
2. Collection title
3. Short collection description
4. Result count and ordering
5. Product grid or predictive results
6. Pagination or fallback discovery action

## Consistency rules

- Uppercase captions belong to context labels and counts only.
- Descriptions stay short and secondary.
- Toolbars should never overpower collection headings.
- Pagination is part of browsing rhythm, not a separate utility block.
