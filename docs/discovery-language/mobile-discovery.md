# Mobile Discovery

## Goal

Mobile discovery should stay calm, reachable, and visually paced.

## Rules

1. Archive toolbars should collapse into a single easy-to-scan column.
2. Search overlays should feel native to thumb reach, with the field and close control immediately available.
3. Predictive results should stack without compressing text or imagery.
4. Empty states need generous vertical spacing so they read as deliberate, not broken.

## Audit focus

- Archive density
- Filter rhythm
- Overlay reachability
- Predictive pacing
- Empty-state spacing
