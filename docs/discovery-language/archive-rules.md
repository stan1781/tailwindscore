# Archive Rules

## Goal

Archive browsing should read as premium editorial discovery, not as a dense catalog utility.

## Rhythm

1. Lead with the collection name and only the amount of context needed to continue browsing.
2. Keep result count and ordering in one restrained toolbar.
3. Let the product grid remain the visual priority.
4. Empty archive states should keep visitors inside collection discovery rather than explaining system behavior.

## Empty-state rules

- Generic empty archives use calm collection language.
- Taxonomy archives use collection-specific language.
- Filtered no-results states should offer a direct path to clear filters.
- Out-of-stock archive states should describe availability, not failure.

## Unsupported patterns

- Dense dashboard toolbars
- Multi-row operator filters above the grid
- Marketing hero copy inside archive discovery
- Plugin-authored fallback wording
