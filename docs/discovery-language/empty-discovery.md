# Empty Discovery

## Goal

Every empty or unavailable discovery state should share one restrained tone.

## Shared tone

- Calm
- Concise
- Premium
- Editorial

## Use this language for

- No products
- Empty collection views
- Filtered no-results
- Out-of-stock archive views
- Search no-results
- Search unavailable states

## Avoid

- Startup cheerleading
- Emoji or mascot copy
- Onboarding instructions
- Plugin fallback phrasing
- Aggressive purchase prompts
