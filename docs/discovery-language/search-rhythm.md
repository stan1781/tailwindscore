# Search Rhythm

## Goal

Search should feel like a quiet extension of collection browsing.

## Default state

1. Start with concise guidance.
2. Offer suggested searches, collection paths, and recent searches.
3. Keep search orientation editorial and product-led.

## Results

- Product matches remain primary.
- Collection paths and suggested refinements remain secondary.
- Predictive results should feel curated, not exhaustive.

## Feedback language

- Loading copy should acknowledge active discovery without sounding technical.
- Unavailable copy should provide a graceful retry.
- No-results copy should widen the browsing path instead of pushing hard calls to action.

## Unsupported patterns

- Algolia-style utility panels
- AI assistant phrasing
- Gamified prompts
- Search dashboards
