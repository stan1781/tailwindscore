# Predictive Search

## Scope

This phase only supports lightweight predictive discovery:

- product title match
- category match
- simple suggestions

## Rules

1. Keep results fast and readable.
2. Limit result count to a calm editorial range.
3. Do not turn predictive search into an analytics or operator console.
4. Fall back to the full search results page for broader exploration.

## Unsupported Behavior

- AI ranking
- semantic retrieval
- merchandising dashboards
- enterprise facet engines
