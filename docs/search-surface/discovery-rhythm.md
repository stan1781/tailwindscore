# Discovery Rhythm

## Goal

Search should keep the browsing experience calm, editorial, and commercially useful.

## Rhythm Rules

1. Keep result groups clearly separated.
2. Product cards inside predictive search should stay lightweight.
3. Suggestions should feel secondary to products.
4. Empty states should guide, not apologize.

## Motion

- use semantic motion tokens
- fade and reveal should stay quiet
- no oversized hover scale
- reduced motion must preserve clarity
