# Mobile Search

## Goal

Mobile search should feel like a premium discovery drawer, not an app screen takeover.

## Rules

- full-height surface is acceptable on mobile
- input should be first focus
- closing should stay simple and predictable
- suggestions should remain readable, not dense
- motion should stay restrained

## Unsupported Behavior

- route-based mobile search screens
- gesture-driven search navigation
- marketplace-density result stacks
