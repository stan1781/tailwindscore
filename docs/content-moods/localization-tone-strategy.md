# Localization Tone Strategy

## Tone-Safe Localization Matrix

| Concern | Preferred Path | Fallback Path |
| --- | --- | --- |
| preserve active mood intent | keep trust, clarity, and action legibility | step down to safer governed surface default |
| avoid abrupt tone loss | avoid compressed transactional wording | use premium commerce default tone |
| avoid inflated tone drift | avoid vague or ceremonial wording | use governed fallback compatible with preset |
| rendering contract | SSR-first and registry-first resolution | never use template-local copy |

## Preset Compatibility Table

| Localization Posture | Meaning |
| --- | --- |
| broadly compatible | active mood is safe across locales |
| conditionally compatible | locale may require governed fallback |
| fallback-prone | use safer governed default early |
