# Supported Surfaces

当前统一纳入 content mood governance 的 surfaces：

- announcement language
- trust messaging
- empty states
- support messaging
- newsletter prompts
- footer messaging
- checkout reassurance
- account messaging
- search guidance

这些 surface 必须保持同一 premium commerce tone family。

如果某个新文案入口会破坏这个统一性，它不应直接进入系统。
