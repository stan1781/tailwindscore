# Mood Rules

TailwindScore 的 content mood 是 registry-driven mood governance。

每个 mood 必须包含：

- `key`
- `label`
- `description`
- `supported surfaces`
- `tone rules`
- `fallback behavior`
- `localization compatibility`

禁止：

- freeform copy systems
- AI copy generation
- uncontrolled marketing language

Content mood 的职责是约束 tone，不是开放任意写作入口。
